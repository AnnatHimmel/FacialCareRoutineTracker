How these files work

  • app_he.arb is the base locale — every key must be defined here.
  • app_he_MA.arb only contains keys where the Moroccan Hebrew text differs from the base. Flutter falls back to app_he.arb for everything else.
  • A key present in app_he_MA.arb but missing from app_he.arb is an orphan — Flutter ignores it.

Start the server with python tools/serve_arb.py to auto-load files and enable direct save.
