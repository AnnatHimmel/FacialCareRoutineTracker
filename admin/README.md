# Skincare Admin Portal

Run locally:
  cd admin
  npm install
  npm start
  Open http://localhost:3001

Usage:
  1. Paste product URLs (YesStyle / OliveYoung / iHerb) — one per line
  2. Click "Fetch & Create Cards"
  3. Edit each card (set category, slots, comment, etc.)
  4. Click "Save master_products.json"
  5. Copy downloaded file to assets/data/master_products.json
  6. Rebuild Flutter app
